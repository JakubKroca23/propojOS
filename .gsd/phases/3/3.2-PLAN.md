# Plan 3.2: Appwrite Plugins Registry

## Objective
Simulate the "Dev Store" by creating an Appwrite Database collection to store registered plugins.

## Context
- .gsd/SPEC.md
- src/lib/appwrite.ts

## Tasks

<task type="auto">
  <name>Create Appwrite Database & Collection Setup Script</name>
  <files>scripts/setupAppwrite.ts, src/lib/appwrite.ts</files>
  <action>
    - Ensure Appwrite SDK is configured for Database (export `Databases` from `appwrite.ts`).

</task>
